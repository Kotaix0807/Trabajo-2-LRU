\doxysubsection{Detailed Description}
Implementacion de las estructuras basicas y funciones para listas ligadas simples. 

Este archivo define los tipos de datos y las operaciones fundamentales para manipular los nodos que componen la estructura de la cache LRU. Se utiliza un nodo cabecera (header node) para la lista.
